Vous pouvez combiner `SourcePicker` avec `ValueDisplay` et `ConfigPanel` via `bind:source` :

```svelte
<script>
  import SourcePicker from "../../../components/SourcePicker.svelte";
  import ValueDisplay from "../../../components/ValueDisplay.svelte";

  let source = {};
</script>

<SourcePicker bind:source />
<ValueDisplay {source} />
```
