---
title: What is a state
description: Explanation of what a state is used for
---

A state is a custom text that you want to use when a value match some condition.

A state is often used to show if the motor is active, or is armed. So you can above or below some thresholds decide to replace the source's value by a custom text. For more complex needs, you can also, by using tags modify the way the source's value is shown.
