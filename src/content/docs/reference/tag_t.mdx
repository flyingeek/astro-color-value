`_t` is the default output used by the color-value widget, as empty state already output the default value, you will only use this tag combined with additional text.

For exemple for a sensor like RxBatt `_t`, will be replaced by `4.15V`, so it is the value with a suffix unit

A common use is when you put a text in a state, it might then be useful to use the combination `_n _t` in the Title State.
