---
title: __ tag
description: The reference for __ tag
---

`__` is one of the simpliest tag to use, it is replaced by a literal underscore

You don't need to write all underscore using this tag: only if it is interpreted as another tag

Exemple you need to use the `_v` suffix, `_v` is a valid tag so in this case you need to write `__v`
