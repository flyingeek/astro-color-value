---
title: __ tag
description: The reference for __ tag
---

`__` is replaced by a literal underscore

You don't need to write all underscore using this tag: only if it is interpreted as another tag

Exemple you want to write `12_v`, `_v` is a valid tag so instead, you have to write `12__v`
