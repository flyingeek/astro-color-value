`_n` is the default output used by the color-value widget in the title. It refers to the source's name.

For exemple for a sensor named RxBatt `_n`, will be replaced by `RxBatt`.

A common use is when you put a text in a state, it might then be useful to use the combination `_n: _t` in the Title State.

<br/>

<ValueDisplay
  source={{
    name: "RxBatt",
    title: "RxBatt: 4.10V",
    stringValue: (v) => "OK"
  }}
/>
